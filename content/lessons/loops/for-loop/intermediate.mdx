# for Loop: Practical Patterns and Techniques

Now that you understand the basics, let's explore practical ways to use `for` loops in real applications.

## Multiple Loop Variables

You can declare and update multiple variables in a single for loop:

```javascript
for (let i = 0, j = 10; i < j; i++, j--) {
  console.log(`i=${i}, j=${j}`);
}
// Output: i=0,j=10 → i=1,j=9 → i=2,j=8 → i=3,j=7 → i=4,j=6
```

This is useful when you need to track two positions - like comparing elements from both ends of an array.

## Common Array Patterns

### Pattern 1: Transform Every Element

```javascript
const numbers = [1, 2, 3, 4, 5];
const doubled = [];

for (let i = 0; i < numbers.length; i++) {
  doubled.push(numbers[i] * 2);
}
console.log(doubled); // [2, 4, 6, 8, 10]
```

### Pattern 2: Find a Specific Item

```javascript
const users = ['Alice', 'Bob', 'Charlie', 'Diana'];
let foundIndex = -1;

for (let i = 0; i < users.length; i++) {
  if (users[i] === 'Charlie') {
    foundIndex = i;
    break; // Exit early once found
  }
}
console.log(foundIndex); // 2
```

### Pattern 3: Sum All Values

```javascript
const prices = [29.99, 9.99, 45.00, 12.50];
let total = 0;

for (let i = 0; i < prices.length; i++) {
  total += prices[i];
}
console.log(`Total: $${total.toFixed(2)}`); // Total: $97.48
```

## Nested Loops

When you put a loop inside another loop, you create a **nested loop**. This is powerful for working with 2D data:

```javascript
// A 3x3 grid
const grid = [
  ['A', 'B', 'C'],
  ['D', 'E', 'F'],
  ['G', 'H', 'I']
];

for (let row = 0; row < grid.length; row++) {
  for (let col = 0; col < grid[row].length; col++) {
    console.log(`Position [${row}][${col}]: ${grid[row][col]}`);
  }
}
```

<InfoBox type="warning" title="Nested Loop Performance">
Nested loops multiply iterations! A loop of 100 inside a loop of 100 = 10,000 iterations. Be careful with large datasets.
</InfoBox>

### Practical Example: Multiplication Table

```javascript
for (let i = 1; i <= 5; i++) {
  let row = '';
  for (let j = 1; j <= 5; j++) {
    row += `${i * j}\t`;
  }
  console.log(row);
}
// Output:
// 1  2  3  4  5
// 2  4  6  8  10
// ...
```

## Loop Directions

### Counting Down (Reverse Loop)

```javascript
for (let i = 5; i > 0; i--) {
  console.log(i);
}
console.log('Liftoff!');
// Output: 5, 4, 3, 2, 1, Liftoff!
```

### Skipping Values

```javascript
// Print only even numbers
for (let i = 0; i <= 10; i += 2) {
  console.log(i);
}
// Output: 0, 2, 4, 6, 8, 10
```

### Iterating Backwards Through an Array

```javascript
const stack = ['first', 'second', 'third'];

for (let i = stack.length - 1; i >= 0; i--) {
  console.log(stack[i]);
}
// Output: third, second, first
```

## Key Patterns Summary

| Pattern | Use Case | Example |
|:--------|:---------|:--------|
| Forward | Default iteration | `i = 0; i < len; i++` |
| Backward | Pop-like operations | `i = len-1; i >= 0; i--` |
| Skip | Every nth element | `i = 0; i < len; i += n` |
| Dual | Compare from both ends | `i = 0, j = len-1; i < j; i++, j--` |
| Nested | 2D grids, combinations | Loop inside loop |
