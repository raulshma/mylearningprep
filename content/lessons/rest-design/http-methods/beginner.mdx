# HTTP Methods & Status Codes: The Language of the Web

When computers talk to each other on the web, they use a specific language. Just like you need verbs to communicate ("get," "create," "delete"), web APIs use **HTTP Methods**. And just like you need feedback ("done!", "error!"), APIs use **Status Codes**.

</InfoBox>
