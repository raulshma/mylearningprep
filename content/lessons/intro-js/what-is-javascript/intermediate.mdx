# JavaScript in Action

JavaScript is a **dynamic, interpreted programming language** that runs directly in your web browser. Unlike languages that need to be compiled (converted to machine code before running), JavaScript code is executed on-the-fly by the browser's JavaScript engine.

## How JavaScript Interacts with Web Pages

JavaScript connects to HTML through the **Document Object Model (DOM)**. Think of the DOM as a tree structure of your web page - JavaScript can walk through this tree and change anything it finds.

```javascript
// Find an element and change its content
const heading = document.querySelector('h1');
heading.textContent = 'Hello, JavaScript!';

// Add interactivity
const button = document.querySelector('button');
button.addEventListener('click', () => {
  alert('Button clicked!');
});
```

## Core Capabilities

JavaScript can:

1. **Manipulate the DOM** - Add, remove, or modify HTML elements
2. **Handle Events** - Respond to user actions (clicks, typing, scrolling)
3. **Make Network Requests** - Fetch data from servers without reloading
4. **Store Data** - Save information in the browser (localStorage, cookies)
5. **Animate Elements** - Create smooth transitions and visual effects

## Event-Driven Programming

JavaScript follows an **event-driven model**. Your code sets up "listeners" that wait for specific events to occur:

```javascript
// Listen for form submission
document.querySelector('form').addEventListener('submit', (event) => {
  event.preventDefault(); // Stop the page from reloading
  const email = document.querySelector('#email').value;
  console.log('Email submitted:', email);
});
```

## Client-Side vs Server-Side

| Client-Side (Browser) | Server-Side (Node.js) |
|:---------------------|:---------------------|
| Runs in user's browser | Runs on a server |
| Manipulates DOM | Handles databases |
| Reacts to user input | Processes API requests |
| Limited access to files | Full system access |

JavaScript's versatility means you can use **one language** for both frontend and backend development!
