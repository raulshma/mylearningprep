# Project Structure

When you create a new ASP.NET Core project, you get a ready-made folder structure. Think of it like moving into a new apartment that already has rooms set up - you just need to know which room is for what!

## Your Project is Like a Well-Organized Kitchen üç≥

Imagine your project as a professional kitchen:

- **The Recipe Book** (`Program.cs`) - The main instructions for how everything runs
- **The Pantry** (`appsettings.json`) - Where you store ingredients (settings)
- **The Prep Stations** (`Controllers/` folder) - Where specific work happens
- **The Storage Room** (`Properties/`) - Behind-the-scenes organization

When you're just starting, focus on these files:

### 1. Program.cs - The Boss üëî

### 2. appsettings.json - The Settings Menu ‚öôÔ∏è

This file stores configuration - like your database address, logging preferences, and other settings your app needs.

### 3. .csproj File - The Shopping List üìù
