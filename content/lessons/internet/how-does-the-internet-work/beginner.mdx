## IP Addresses: Every Device Has an Address

Just like your home has a street address, **every device on the Internet has a unique address** called an **IP Address**.

<InfoBox type="fun-fact">
  IP stands for "Internet Protocol" - it's like the language computers use to
  find each other!
</InfoBox>

<InteractiveDemo type="ip-address-demo" />

### What Does an IP Address Look Like?

An IP address looks like four numbers separated by dots:

```
192.168.1.100
```

Each number can be from 0 to 255. This gives us billions of possible addresses!

## DNS: The Internet's Phone Book

You probably don't remember IP addresses like `172.217.14.206`. Instead, you remember names like `google.com`. But computers need IP addresses to connect!

The **Domain Name System (DNS)** translates friendly names into IP addresses.

<AnimatedDiagram type="dns-lookup" />

### How DNS Works (Simple Version)

1. You type `google.com` in your browser
2. Your computer asks a DNS server: "What's the IP for google.com?"
3. The DNS server responds: "It's 172.217.14.206"
4. Your browser connects to that IP address

<InfoBox type="tip">
  Think of DNS like your phone's contact list. You don't memorize phone numbers
  - you just tap on "Mom" and your phone knows the number!
</InfoBox>
